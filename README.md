A comprehensive data analysis project exploring **Indian Premier League (IPL) match data from 2008 to 2025**, uncovering team performance trends, win statistics, and insightful patterns using Python, Pandas, and data visualization libraries.
 Project Overview
The IPL is one of the most popular T20 cricket leagues in the world. This project dives deep into historical IPL match data to answer questions like:
- Which teams have dominated the IPL over the years?
- What are the most common winning margins?
- How does toss decision impact match outcomes?
- Which venues have hosted the most matches?
- How have team performances evolved season by season?
 Repository Structure

├── IPL_Data_Analysis.ipynb   # Main Jupyter Notebook with full analysis

 Key Analyses Performed
- **Team Performance:** Win counts, win percentages, and season-wise performance for all franchises
- **Toss Analysis:** Impact of toss decisions (bat/field) on match outcomes
- **Venue Statistics:** Most-used stadiums and their win-rate breakdown
- **Win Type Distribution:** Wins by runs vs. wins by wickets
- **Season Trends:** Year-over-year performance shifts across IPL seasons (2008–2025)
- **Top Performers:** Most successful teams across the entire IPL history

 Getting Started
 Prerequisites
Make sure you have Python 3.x installed. Then install the required libraries:

```bash
pip install pandas matplotlib seaborn jupyter
```
 Running the Notebook
1. Clone the repository:
   ```bash
   git clone https://github.com/warrior678/IPL_cricket_matches.git
   cd IPL_cricket_matches
   ```
2. Launch Jupyter Notebook:
   ```bash
   jupyter notebook IPL_Data_Analysis.ipynb
   ```
   3. Run all cells to reproduce the analysis and visualizations.

Sample Insights
- **Most Successful Team:** Mumbai Indians have consistently led in overall wins across IPL history.
- **Toss Advantage:** Teams winning the toss and choosing to field have a slight edge in match outcomes.
- **High-Scoring Venues:** Certain stadiums show consistently higher average scores, influencing team strategies.

 Dataset

The dataset covers **IPL matches from 2008 to 2025** and includes:
- Match ID, Season, Date
- Teams (Team 1 vs Team 2)
- Toss winner and decision
- Match result and winner
- Win margin (runs or wickets)
- Venue and City
- Player of the Match
